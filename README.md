# LUMENFALL

LUMENFALL is a custom C++ / OpenGL voxel game prototype with procedural terrain, long-distance chunk streaming, real-time shadows, water, foliage, post-processing, and large-scale world rendering.

This build is a Windows executable demo.

## Features

- Procedural voxel world generation
- Huge render distances with LOD chunk streaming
- Real-time cascaded shadows
- Water, foliage, bloom, SSAO, TAA, MSAA, and HDR-style rendering
- Survival-style world with saves, entities, UI, and settings
- No external asset pack required; most visuals are generated by the engine

## Download

Grab the latest Windows build from the Releases page.

Unzip it and run `lumenfall.exe`.
